Program ini merupakan program Control Drone secara otomatis untuk mengikuti objek tertentu dan menuju kepada objek tertentu yang dipilih dari camera. 

program ini sudah di uji dengan menggunakan Raspberri pi dengan FC Pixhawk6C dengan meggunakan firmware ardupilot.
